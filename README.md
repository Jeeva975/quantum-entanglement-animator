# 🌌 Quantum Entanglement Animator

A 3D quantum circuit simulator built with React, FastAPI, and Qiskit.

## 🚀 Local Setup Instructions

### 1. Clone the Project
```bash
git clone https://github.com/Jeeva975/quantum-entanglement-animator.git
cd quantum-entanglement-animator

#Backend setup
cd backend
python -m venv venv

# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

pip install -r requirements.txt
python app/main.py

#Frontend Setup
Open a new terminal:

cd frontend
npm install
npm run dev

#Finallly open 
http://localhost:5173 in your browser
